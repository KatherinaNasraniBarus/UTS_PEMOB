### Deskripsi Aplikasi

Aplikasi ini memungkinkan pengguna untuk memasukkan nama mereka, memilih kategori kuis (saat ini C++ atau Dart), dan menjawab serangkaian pertanyaan pilihan ganda.

**Fitur Utama:**

* **Validasi Nama :** Pengguna harus memasukkan nama sebelum memulai kuis.
* **Pemilihan Kategori :** UI yang bersih untuk memilih antara kuis C++ atau Dart.
* **UI Responsif :** Tampilan beradaptasi dengan baik pada berbagai ukuran layar (dari ponsel kecil hingga tablet).
* **Sistem Kuis :** Menampilkan satu pertanyaan pada satu waktu dengan opsi pilihan ganda.
* **Perhitungan Skor :** Skor dihitung berdasarkan jawaban yang benar (setiap jawaban benar bernilai 20 poin).
* **Dialog Hasil :** Menampilkan skor akhir setelah kuis selesai dengan opsi untuk kembali ke layar utama.

### Credit Aset & Dependensi

Aplikasi ini menggunakan beberapa aset eksternal dan dependensi:

1.  **Dependensi (Dependencies):**
    * `flutter`: Dependensi inti dari Flutter SDK.
    * `google_fonts: ^6.2.1`: Digunakan untuk memuat font "Poppins" dari Google Fonts.(Sumber: [fonts.google.com](https://fonts.google.com/specimen/Poppins))
    * `cupertino_icons: ^1.0.8`: Menyediakan set ikon gaya iOS (Cupertino).

2.  **Dependensi Pengembangan (Dev Dependencies):**
    * `flutter_test`: Menyediakan alat untuk *automated testing*.
    * `flutter_lints: ^5.0.0`: Menganalisis kode untuk menemukan error dan menjaga gaya koding.
